💻📚 Book Store Management System

🚀 A modern web-based application built using Node.js, Express.js, MongoDB, and EJS that helps users easily manage book records with a smooth and stylish interface.

🎯 Objective
💡 To develop a system where users can add, view, update, and delete books efficiently while maintaining all data in a database with image support.

🧩 Key Features
📚 Add New Book
➕ Enter book details like title, author, category, price, quantity, and description
🖼️ Upload book cover image using Multer
✅ Store everything securely in database

📖 View All Books
👀 Display all books in a clean card-based UI
📸 Show book image, title, author, category, price, and quantity
🎨 Modern responsive design with smooth layout

✏️ Update Book
🔄 Edit existing book details anytime
📝 Modify title, price, quantity, or description
🖼️ Option to update book image

❌ Delete Book
🗑️ Remove unwanted books from database
⚡ Quick delete using book ID with confirmation

🖼️ Image Management
📂 Upload images using Multer
💾 Store image paths in MongoDB
🖥️ Display images properly on frontend

📊 Dashboard Summary
📈 View total number of books available
📚 Quick overview of inventory

🛠️ Technologies Used
💻 Backend: Node.js + Express.js
🗄️ Database: MongoDB (Mongoose)
🎨 Frontend: EJS Templates
📤 File Upload: Multer
⚡ Development Tool: Nodemon

✨ Highlights
🔥 Full CRUD functionality
🎯 Simple and user-friendly interface
📱 Responsive “vibe-coded” UI
🧠 Clean MVC architecture
⚡ Fast and efficient data handling

📌 Example Output

🎯 Conclusion
💡 This project demonstrates how to build a complete full-stack CRUD application with real-world features like image upload, database integration, and dynamic UI — making it perfect for learning modern web development 🚀
